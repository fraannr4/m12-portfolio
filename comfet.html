Resum del Portafoli

Aquesta pàgina web és un portafoli personal dissenyat per destacar l'experiència, habilitats i projectes de Francesco Romano

Utilitza tecnologies web estàndard com HTML, CSS i un enfocament estructurat i estilitzat. A continuació, et faig un resum de com s'ha construït:

Estructura HTML

  • Capçalera (<header>): Inclou un títol i un menú de navegació amb enllaços a les seccions principals de la pàgina (Inici, Sobre Mi, Habilitats, Projectes i Contacte).
  • Contingut principal (<main>): Es divideix en diverses seccions:
    • Inici: Introducció de l'usuari amb un fons atractiu i un enllaç que convida a contactar.
    • Sobre Mi: Presenta un resum professional i la formació acadèmica de Francesco.
    • Habilitats: Es destaquen les habilitats dividides en àrees com el desenvolupament web, les bases de dades, l'administració de sistemes i DevOps.
    • Projectes: Mostra els projectes destacats, cadascun amb una imatge, un títol i una breu descripció.
    • Contacte: Inclou un formulari amb camps per nom, correu electrònic i missatge.
  • Peu de pàgina (<footer>): Conté enllaços a xarxes socials, drets d'autor i un enllaç per veure com s'ha fet la pàgina.

Estils CSS

  • Disseny visual: Fons fosc amb text clar per crear una estètica moderna i professional.
  • Tipografia llegible i consistent: Utilitzant la font 'Segoe UI'.
  • Disseny adaptable: Es fa servir CSS Grid i Flexbox per crear un disseny flexible que s'adapti a diferents mides de pantalla.
  • Interactivitat: Efectes hover per enllaços i elements interactius (com els projectes i botons), millorant l'atractiu visual.
  • Transicions suaus: En el canvi de colors o quan els elements s'amplien lleugerament.

Recursos externs

  • S'importa una fulla d'estils externa (styles.css) per mantenir el codi organitzat i reutilitzable.

Aspectes tècnics

  • S'utilitza una estructura semàntica amb etiquetes com <nav>, <main>, <section>, <footer>, millorant l'accessibilitat i el SEO de la pàgina.
  • S'utilitzen propietats modernes de CSS com background: cover, box-sizing i transition.

Enllaços externs

  • Al final de la pàgina, es poden veure enllaços a les xarxes socials com Twitter, LinkedIn, GitHub i CodePen, permetent que el visitant es posi en contacte amb el desenvolupador a través d'aquestes plataformes.

Resum

En resum, aquest portafoli està dissenyat per ser funcional, atractiu i fàcil de navegar, mostrant eficaçment les capacitats i projectes del desenvolupador.
